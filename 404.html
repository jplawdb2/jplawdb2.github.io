Filename Rules

Article "の" → underscore "_"
  措法66の6  → text/sochi/66_6.txt
  法法22の2  → text/hojin/22_2.txt
Circular dashes are kept as-is:
  法基通2-1-1 → text/hojin-kihon/2-1-1.txt

Common Mistakes

✗ text/sochi/66-6.txt    (hyphen in law article)
✓ text/sochi/66_6.txt    (underscore)

✗ text/hojin_kihon/...   (underscore in code name)
✓ text/hojin-kihon/...   (hyphen in code name)
